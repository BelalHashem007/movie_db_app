# movie_db_app
- A Movie db where users can find their next movie or create their watchlist. it includes reviews, rating, actors and anything you need to know about the movie.

- Mainly a side project to practise TypeScript and Redux also tailwindcss.


## Features
- User can see all the latest movie on the front page.
- User scroll down to see all other movies according to release date.
- User can click on any of the movie to go to their own separate page.
- User can then see all about the movie ratings, about, actors present on each separate movie page.
- User can create an account.
- User can create their own watch list.
- User can review movies.
